# Tools

## godoc (alias of 'go doc')

![Demo: godoc](resources/godoc.gif)

### Usage

```sh
# View GoDoc of 'net/http' package 
godoc http

# View GoDoc of 'Server' in 'net/http' package 
godoc http.Server
```

**Shortcut Key**

- Ctrl-o: Open web page(http://godoc.org) of GoDoc of package in browser
